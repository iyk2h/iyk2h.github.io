### import

```java
import java.util.Scanner;
```



### nextLine()

입력한 값을 문자열로 반환하는 메서드. 줄바꿈 문자열 단위로 반환

```java
Scanner scanner = new Scanner(System.in);

String value = scanner.nextLine();
```



### nextInt(), nextShort(), nextByte(), nextLong() -> 정수 반환

### nextDouble() -> 실수 반환

### nextBoolean() -> true or false 반환
